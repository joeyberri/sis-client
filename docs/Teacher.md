# Teacher

Purpose: day-to-day classroom work — grade entry, assignments, attendance, student insights.

## Sidebar

-   Dashboard
    
-   My Classes
    
    -   Class Roster (per class)
        
    -   Gradebook
        
    -   Attendance
        
    -   Assignments
        
-   Student Profiles (quick search)
    
-   Assessments
    
-   Analytics (class-level)
    
-   Saved Views
    
-   Messages (to parents / staff)
    
-   Resources (templates, rubrics)
    
-   Help / Tutorials
    

* * *

### Dashboard (page)

-   Today’s classes (schedule)
    
-   Tasks: ungraded submissions, missing marks, upcoming exams
    
-   Quick shortcuts: Enter grades, Mark attendance, Send parent note
    

### My Classes → Class Roster (page)

-   Virtualized roster list: Photo | Name | ID | Attendance today | Avg Grade | Alerts
    
-   Quick actions per row: Mark present/absent, Open grade entry, Add behavior note, Message parent
    

### My Classes → Gradebook (page)

-   Spreadsheet-like AG Grid with columns: Student | Assignment1 | QuizA | Midterm | Final | Term Avg | Letter
    
-   Inline editing with autosave
    
-   Validation rules (min/max), highlighting outliers
    
-   Controls: Export CSV, Apply grade curve, Bulk enter, Restore from snapshot
    
-   Sidebar: class-level stats (avg, median, distribution histogram) and quick filter (missing scores, below threshold)
    

### My Classes → Attendance

-   Fast check-in UI (single-click present/absent)
    
-   Bulk mark options
    
-   Excuse reason modal
    
-   Send parent absent notification button
    

### Assignments

-   List of published assignments with status (published/draft)
    
-   Create assignment wizard (use Tiptap)
    
-   Collect submissions list; grading workflow (mark as graded, leave feedback)
    
-   Plagiarism check status (if enabled)
    

### Student Profiles (teacher view)

-   Snapshot card: photo, class, guardians, contact
    
-   Tabs: Grades (timeline), Attendance (timeline), Behavior notes, Interventions, Files
    
-   Action buttons: Message parent, Add intervention, Flag for counseling
    

### Analytics (class-level)

-   Trend chart: class average over time
    
-   Distribution heatmap of scores by assessment
    
-   Student ranking (real-time) with freeze snapshot option
    
-   Alerts specific to class (declines, missing submissions)
    

### Messages

-   One-to-one messaging with parent or student (if allowed)
    
-   Templates: missing assignment notice, congratulations, request meeting
