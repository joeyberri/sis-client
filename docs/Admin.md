### Dashboard (page)

-   KPIs: student count, attendance today (%), average grade (term), alerts active
    
-   Quick actions: Enroll student, Create class, Snapshot ranking, Send parent digest
    
-   Charts: attendance trend, grade distribution, top declines
    

### Students → All Students (page)

-   Table columns: Photo | Name | Student ID | Class | Status | Attendance % | Avg Score | Actions
    
-   Filters: year, grade, status, guardian name
    
-   Row actions: View profile, Edit, Promote, Archive
    
-   Bulk actions: Bulk enroll, CSV import
    

### Students → Enroll Student

-   Form: basic details, guardians, initial enrollment class, documents upload
    
-   Validation & duplicate detection
    

### Teachers → All Teachers

-   Table: Name | ID | Subjects | Classes | Email | Status
    
-   Actions: Assign to class, View workload, Reset password
    

### Classes & Courses → All Classes

-   Table: Class name | Subject | Teacher(s) | Students | Schedule
    
-   Row: Manage roster, open gradebook, timetable link
    

### Timetable Builder

-   FullCalendar interface
    
-   Conflict resolution modal (teacher/room conflicts)
    
-   Publish/unpublish schedule
    

### Assessments & Grading → Create Assessment

-   Form: title, class/course, date, weight, grading scale (select or custom), rubric attach
    
-   Quick preview of how scoring affects term grade
    
-   Save & schedule reminder for teachers to grade
    

### Grade Configs

-   List of per-course configs (JSON editor + guided form)
    
-   Actions: Add new, Clone, Apply to class, Rollback to previous version
    

### Gradebook Overview

-   Class list showing completion % of grading, average grades
    
-   Quick snapshot export (CSV/PDF)
    
-   Button: Create ranking snapshot (immutable)
    

### Attendance

-   Today's attendance quick grid (virtualized list)
    
-   Filters: class, date
    
-   Bulk mark, excuse reason, send parent notification for absences
    
-   Attendance history per student
    

### Reports → Report Cards

-   Generate report cards (template selection)
    
-   Preview PDF, bulk generate for term
    
-   Schedule automatic email distribution to guardians
    

### Analytics & Insights

-   Saved views list
    
-   Prebuilt views: Top declines, At-risk students, Teacher performance
    
-   Ability to open Metabase embedded dashboards
    

### Alerts & Automations

-   View active alerts
    
-   Create automation rule (IF condition → THEN action)
    
-   Alert history and resolution
    

### Parents & Guardians

-   List of guardians, linked students
    
-   Invite/resend access
    
-   Parent portal settings (what parents see)
    

### Fees & Payments

-   Fee schedules, invoice list, payment status
    
-   Send dues reminders
    

### Documents

-   Shared templates, enrollment forms, policies
    
-   Set permissions (teacher, parent, admin)
    

### Settings → Users & Permissions

-   Role assignment per user
    
-   Permission overrides for specific users
