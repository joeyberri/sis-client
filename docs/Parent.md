# Parent

Purpose: view-only access to their child(ren)’s progress and communication.

## Sidebar

-   Dashboard
    
-   My Children
    
    -   Child Profile (per child)
        
        -   Grades
            
        -   Attendance
            
        -   Assignments
            
        -   Report Cards
            
-   Notifications
    
-   Messages
    
-   Billing (if applicable)
    
-   Settings
    
    -   Notification preferences
        
    -   Linked accounts
        
-   Help
    

* * *

### Dashboard (page)

-   Snapshot cards per child: latest grade, attendance today, upcoming assignments, alerts (at-risk, missing work)
    
-   Quick actions: Message teacher, Request meeting
    

### Child Profile → Grades

-   Current term grades: courses listed with numeric + letter + trend sparkline
    
-   Downloadable report card PDF
    
-   Interpretive help: what grade bands mean, coach tips
    

### Child Profile → Attendance

-   Calendar view with absences highlighted
    
-   Summary: % present this term, recent late marks
    
-   Request absence excuse (upload doc)
    

### Child Profile → Assignments

-   List of upcoming & past assignments
    
-   Submission status, teacher feedback
    
-   Ability to download submission attachments
    

### Notifications

-   Digest options (daily/weekly)
    
-   Notification history (messages, alerts)
    

### Messages

-   Conversation threads with teachers
    
-   Templates for common messages
